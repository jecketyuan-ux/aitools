## 1. 系统简介

### 1.1 什么是EduFlow
EduFlow是一个企业级在线培训平台，为企业提供完整的员工培训解决方案。系统支持课程管理、学员管理、学习进度跟踪等功能。

### 1.2 系统特点
- **多角色管理**: 支持管理员和学员不同角色
- **课程管理**: 完整的课程创建、发布、管理流程
- **学习跟踪**: 详细的学习进度和时长统计
- **资源管理**: 支持视频、图片、文档等多种资源
- **部门管理**: 支持企业组织架构管理
- **权限控制**: 基于角色的权限管理

### 1.3 系统要求
- **浏览器**: Chrome 80+、Firefox 75+、Safari 13+、Edge 80+
- **网络**: 稳定的互联网连接
- **设备**: PC、平板、手机均可访问

## 2. 管理员使用指南

### 2.1 登录系统

#### 2.1.1 访问地址
```
管理后台地址: http://your-domain.com/backend
```

#### 2.1.2 登录步骤
1. 打开浏览器，访问管理后台地址
2. 输入管理员邮箱和密码
3. 点击"登录"按钮
4. 登录成功后进入管理控制台

#### 2.1.3 默认管理员账户
```
邮箱: admin@eduflow.com
密码: eduflow123
```

⚠️ **安全提示**: 首次登录后请立即修改默认密码

### 2.2 用户管理

#### 2.2.1 查看用户列表
1. 在左侧菜单点击"用户管理"
2. 系统显示所有用户列表
3. 可按用户名、邮箱、状态等条件筛选

#### 2.2.2 创建新用户
1. 点击"添加用户"按钮
2. 填写用户信息：
   - 邮箱地址（必填，作为登录账号）
   - 姓名（必填）
   - 密码（可选，不填则系统自动生成）
   - 所属部门
   - 身份证号（选填）
3. 点击"保存"完成创建

#### 2.2.3 编辑用户信息
1. 在用户列表中找到目标用户
2. 点击"编辑"按钮
3. 修改用户信息
4. 点击"保存"确认修改

#### 2.2.4 用户状态管理
- **激活用户**: 点击"激活"按钮，用户可正常登录
- **锁定用户**: 点击"锁定"按钮，用户无法登录
- **重置密码**: 为用户生成新密码或发送重置邮件

### 2.3 部门管理

#### 2.3.1 部门结构
- 支持多级部门结构
- 可设置父子部门关系
- 支持部门排序

#### 2.3.2 创建部门
1. 在左侧菜单点击"部门管理"
2. 点击"添加部门"
3. 填写部门信息：
   - 部门名称
   - 上级部门（可选）
   - 排序值
4. 点击"保存"

#### 2.3.3 部门用户分配
1. 选择目标部门
2. 点击"分配用户"
3. 从用户列表中选择要分配的用户
4. 确认分配

### 2.4 课程管理

#### 2.4.1 创建课程
1. 在左侧菜单点击"课程管理"
2. 点击"创建课程"
3. 填写课程基本信息：
   - 课程标题（必填）
   - 课程简介
   - 课程封面图片
   - 课程类型（必修/选修）
   - 学时数
   - 课程分类
4. 点击"下一步"

#### 2.4.2 添加章节
1. 在课程编辑页面点击"添加章节"
2. 输入章节名称
3. 设置章节排序
4. 点击"保存"

#### 2.4.3 添加课时
1. 选择目标章节
2. 点击"添加课时"
3. 填写课时信息：
   - 课时标题
   - 课时类型（视频/文档/音频）
   - 关联资源文件
   - 预计学习时长
4. 点击"保存"

#### 2.4.4 课程发布
1. 完成课程内容编辑后
2. 点击"发布课程"
3. 设置发布范围：
   - 全员可见
   - 指定部门
   - 指定用户
4. 确认发布

#### 2.4.5 学习进度查看
1. 在课程列表中选择目标课程
2. 点击"学习统计"
3. 查看以下数据：
   - 总学习人数
   - 完成人数
   - 平均完成率
   - 学习时长统计
   - 个人学习详情

### 2.5 资源管理

#### 2.5.1 上传视频资源
1. 在左侧菜单点击"资源管理"
2. 点击"上传视频"
3. 选择视频文件（支持MP4、AVI、MOV等格式）
4. 填写视频信息：
   - 视频标题
   - 视频描述
   - 视频分类
5. 点击"上传"

#### 2.5.2 上传图片资源
1. 点击"上传图片"
2. 选择图片文件（支持JPG、PNG、GIF等格式）
3. 填写图片信息
4. 点击"上传"

#### 2.5.3 资源分类管理
1. 点击"分类管理"
2. 创建资源分类
3. 为资源分配分类
4. 支持多级分类

#### 2.5.4 资源使用统计
1. 查看资源列表
2. 查看每个资源的：
   - 下载次数
   - 播放次数
   - 使用时长
   - 关联课程数

## 3. 学员使用指南

### 3.1 登录系统

#### 3.1.1 访问地址
```
学习平台地址: http://your-domain.com
```

#### 3.1.2 用户注册
1. 访问学习平台地址
2. 点击"注册"按钮
3. 填写注册信息：
   - 邮箱地址
   - 姓名
   - 密码
   - 确认密码
4. 点击"注册"完成账号创建

#### 3.1.3 用户登录
1. 输入邮箱和密码
2. 点击"登录"
3. 登录成功后进入学习平台

### 3.2 个人信息管理

#### 3.2.1 查看个人信息
1. 点击右上角头像
2. 选择"个人中心"
3. 查看个人信息：
   - 姓名
   - 邮箱
   - 所属部门
   - 注册时间
   - 最后登录时间

#### 3.2.2 修改个人信息
1. 在个人中心点击"编辑资料"
2. 修改可编辑的信息
3. 点击"保存"确认修改

#### 3.2.3 修改密码
1. 在个人中心点击"修改密码"
2. 输入当前密码
3. 输入新密码
4. 确认新密码
5. 点击"确认修改"

### 3.3 课程学习

#### 3.3.1 浏览课程
1. 登录后进入"我的课程"页面
2. 查看可用课程列表：
   - 全部课程
   - 必修课程
   - 选修课程
   - 已完成课程
   - 学习中课程

#### 3.3.2 开始学习
1. 点击目标课程
2. 进入课程详情页面
3. 查看课程信息：
   - 课程简介
   - 课程大纲
   - 学习要求
   - 学时信息
4. 点击"开始学习"

#### 3.3.3 学习课程内容
1. 按章节顺序学习课程内容
2. 每个课时完成后点击"完成"
3. 系统自动记录学习进度
4. 支持以下学习模式：
   - 视频学习
   - 文档阅读
   - 音频收听

#### 3.3.4 学习进度跟踪
1. 在课程页面查看学习进度：
   - 总进度百分比
   - 已完成章节数
   - 已完成课时数
   - 累计学习时长
2. 在个人中心查看：
   - 所有课程的学习进度
   - 学习时长统计
   - 学习完成率

#### 3.3.5 继续学习
1. 在课程列表中找到"学习中"的课程
2. 点击"继续学习"
3. 系统自动跳转到上次学习位置
4. 继续未完成的学习内容

### 3.4 学习证书

#### 3.4.1 获得证书
- 完成必修课程后自动获得学习证书
- 证书包含：
  - 课程名称
  - 完成时间
  - 学习时长
  - 学员姓名

#### 3.4.2 查看证书
1. 在个人中心点击"我的证书"
2. 查看已获得的所有证书
3. 点击"查看详情"查看证书内容
4. 支持"下载证书"功能

### 3.5 学习统计

#### 3.5.1 个人学习数据
1. 在个人中心查看学习统计：
   - 总学习时长
   - 完成课程数
   - 正在学习课程数
   - 本月学习时长
   - 学习连续天数

#### 3.5.2 学习报告
1. 点击"学习报告"
2. 查看详细学习数据：
   - 每日学习时长
   - 课程完成情况
   - 学习效率分析
   - 学习建议

## 4. 常见问题

### 4.1 登录问题

**Q: 忘记密码怎么办？**
A: 请联系管理员重置密码，或使用"忘记密码"功能（如已开启）。

**Q: 登录时提示"账号被锁定"怎么办？**
A: 账号被锁定通常是由于多次输错密码，请联系管理员解锁。

**Q: 无法访问系统怎么办？**
A: 请检查网络连接，或联系IT部门确认系统状态。

### 4.2 学习问题

**Q: 视频无法播放怎么办？**
A: 请检查网络连接，或尝试更换浏览器。如仍有问题，请联系管理员。

**Q: 学习进度没有保存怎么办？**
A: 请确保每个课时学习完成后点击"完成"按钮。如进度仍不正确，请联系管理员。

**Q: 课程学习完成后没有获得证书怎么办？**
A: 请确认已完成所有必修课时，如仍有问题，请联系管理员。

### 4.3 技术问题

**Q: 系统运行缓慢怎么办？**
A: 请检查网络连接，或尝试在非高峰时段使用。

**Q: 文件上传失败怎么办？**
A: 请检查文件大小和格式是否符合要求，或联系管理员。

**Q: 移动端如何使用？**
A: 系统支持响应式设计，可在手机浏览器中正常使用。
